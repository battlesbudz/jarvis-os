# OpenClaw Research Findings

## Overview
OpenClaw is a self-hosted AI assistant that connects various messaging platforms to coding agents. It runs on your own machine and is open source under the MIT License.

### Documentation Links:
- [OpenClaw Official Documentation](https://docs.openclaw.ai/)
- [OpenClaw on DigitalOcean](https://docs.digitalocean.com/products/marketplace/catalog/openclaw/)
- [OpenClaw Tutorials on YouTube](https://www.youtube.com/watch?v=st534T7-mdE)

### GitHub Information:
OpenClaw is hosted on GitHub and encourages community involvement. Over 365k stars on GitHub. It integrates with GitHub for automated workflows and contributions.
- [OpenClaw GitHub Repository](https://github.com/openclaw/openclaw)
- [OpenClaw GitHub Integration](https://clawbot.ai/wiki/industry/openclaw-github-integration.html)

### Project Features:
- **Integration with APIs**: OpenClaw can connect to various services allowing real-time interaction.
- **Communication Platforms**: Supports multiple messaging platforms with built-in plugins.
- **Local Orchestration**: Provides a local layer to interact with AI models.
- **Mobile Nodes**: Capability for on-the-go access.
