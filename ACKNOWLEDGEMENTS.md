# Acknowledgements & Inspirations

Jarvis OS builds on ideas from the broader open-source agent, memory, automation, and personal AI ecosystem. This file records public attribution for projects that influenced or contributed to Jarvis's architecture.

## GBrain

Parts of Jarvis OS's memory and knowledge-management architecture are derived from and inspired by GBrain, an open-source project created by Garry Tan:

https://github.com/garrytan/gbrain

Jarvis adapts those concepts into its own Memory OS architecture. In Jarvis, the GBrain-inspired layer is used as a derived, rebuildable memory/index layer over canonical user memory, provenance, review, and correction flows rather than as a standalone replacement for Jarvis's source-of-truth memory tables.

Related Jarvis docs:

- `docs/gbrain-implementation-plan.md`
- `docs/gbrain-spec-sheet.md`
