# Research Context

## Purpose
Structured repository for topic-based research and citation-backed analysis.

## When to Use
Use for investigation, evidence gathering, competitor analysis, legal/grant exploration, and synthesis.

## Read First
- Topic folder matching request (`ai-agents/`, `cannabis/`, `legal/`, etc.)
- Existing dated research briefs before creating new ones

## Output Types
Dated research briefs, annotated findings, source/citation notes, recommendations.

## Do Not Load By Default
Do not load unrelated workspace domains unless the question requires cross-domain synthesis.
