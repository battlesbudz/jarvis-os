# Content Studio Context

## Purpose
End-to-end content workspace from idea capture through drafts, finals, and repurposing.

## When to Use
Use for content planning, script writing, email content, social posts, and multi-channel repurposing.

## Read First
- `ideas/` for source concepts
- Most recent drafts in `drafts/` or scripts in `scripts/`

## Output Types
Scripts, drafts, finalized content, email variants, social/post packages.

## Do Not Load By Default
Do not load business/legal folders unless content requires compliance or brand constraints.
