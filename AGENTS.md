# AGENTS.md

This file is the workflow and tool-usage index for Jarvis agents.

- Personality lives in `SOUL.md`.
- The master orchestrator contract lives in `agents/PRIME.md`.
- Routing lives in `agents/ROUTING.md`.
- Context-loading rules live in `agents/CONTEXT.md`.
- Tool safety and approval policy live in `agents/TOOL_POLICY.md`.
- Crew-specific behavior lives in `agents/crew/`.

Keep workflow rules here or in the referenced agent files, not in `SOUL.md`.
